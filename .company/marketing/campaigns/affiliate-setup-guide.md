## 1. アフィリエイトプログラム登録手順

---

### 1-1. Feel（フィール）

| 項目 | 内容 |
|------|------|
| ASP名 | Link-A（リンクエー） |
| 登録URL | https://www.link-a.net/ |
| 報酬単価 | 18,639円（新規有料会員登録1件につき） |
| 審査 | あり（通常3〜7営業日） |
| 報酬発生条件 | ユーザーがアフィリエイトリンク経由でFeelに新規登録し、有料サービスを利用した時点 |

#### 登録手順

1. Link-A（https://www.link-a.net/）にアクセス
2. 「無料会員登録」をクリックし、メールアドレス・パスワード等を入力して登録
3. 登録後、管理画面にログインし「プログラム検索」から「Feel」を検索
4. 「提携申請」ボタンを押し、サイト情報（Instagram URL / TikTok URL / LINE運用概要）を入力して申請
5. 審査通過後、管理画面から専用アフィリエイトリンクを発行
6. リンクをInstagram・TikTok・LINEの投稿に設置

#### 注意事項

- 審査時に「媒体URL」の入力が必須。InstagramとTikTokのプロフィールURLを記載する
- 「リスト広告（LINE配信）」の利用可否は提携条件に明記されていない場合があるため、審査通過後に担当者へメールで確認すること
- 虚偽・誇大表現（「必ず復縁できる」等）はNG。ASP規約・景品表示法に従った表現を使用すること
- Cookie有効期間（アトリビューション期間）をLink-A管理画面で確認し、投稿とリンクのタイミングを合わせること

### 1-2. ピュアリ（Pureri）

| 項目 | 内容 |
|------|------|
| ASP名 | A8.net（エーハチネット） |
| 登録URL | https://www.a8.net/ |
| 報酬単価 | 15,000円（新規有料会員登録1件につき） |
| 審査 | あり（通常3〜5営業日） |
| 報酬発生条件 | アフィリエイトリンク経由でピュアリに新規登録し、有料ポイントを購入した時点 |

#### 登録手順

1. A8.net（https://www.a8.net/）にアクセスし「メディア（サイト運営者）として登録」を選択
2. メールアドレス・パスワード・サイト情報を入力して会員登録
3. 管理画面「プログラム」タブから「ピュアリ」を検索
4. 「提携申請」ボタンを押し、メディア情報（SNS URL・配信媒体の説明）を入力して申請
5. 審査通過後、プログラム管理画面から広告リンクを発行（テキストリンク推奨）
6. 各SNS・LINE配信に設置

#### 注意事項

- A8.netはSNSメディアとしての登録が可能。登録時に「メディア種別」で「SNS（Instagram / TikTok）」を選択する
- LINE配信での利用はA8.net規約上「メールマガジン広告」に該当する可能性があるため、事前にA8サポートへ確認することを推奨
- 成果承認は広告主（ピュアリ側）が行うため、承認まで最大2〜3ヶ月かかる場合がある
- スクリーンショット等による不正誘導は厳禁

### 1-3. ヴェルニ（Vernis）

| 項目 | 内容 |
|------|------|
| ASP名 | afb（アフィリエイトB） |
| 登録URL | https://www.afb.ne.jp/ |
| 報酬単価 | 11,000円（新規有料会員登録1件につき） |
| 審査 | あり（通常5〜10営業日） |
| 報酬発生条件 | アフィリエイトリンク経由でヴェルニに新規登録し、有料サービスを利用した時点 |

#### 登録手順

1. afb（https://www.afb.ne.jp/）にアクセスし「パートナー登録」をクリック
2. 必要事項（氏名・住所・振込先口座・メディア情報）を入力して登録
3. 登録完了後、管理画面の「プログラム検索」から「ヴェルニ」を検索
4. 「提携申請」を行い、媒体情報（Instagram・TikTok URL、LINE配信の概要）を記載
5. 審査通過後、広告リンクを発行し各チャネルに設置

#### 注意事項

- afbは審査が比較的厳しめ。InstagramとTikTokの投稿実績（フォロワー数・投稿数）が審査材料になるため、プロフィールを整えた状態で申請すること
- 「当サービスを利用して復縁できた」等の体験談を装った虚偽投稿はNG（ステルスマーケティング防止法に抵触する恐れあり）
- 必ずPR表記（「#PR」「#広告」「#アフィリエイト」）を投稿に明記すること（景品表示法・ステマ規制対応）

---

## 2. 共通注意事項

- **ステマ規制対応（2023年10月施行）**: すべての投稿にPR・広告の明示が必要。Instagram・TikTokでは投稿文に「#PR」または「#広告」を記載すること
- **報酬の確定タイミング**: 各ASPで「仮承認 → 確定承認」のフローがある。確定承認前に成果がキャンセルされる場合があるため、月次で管理画面を確認すること
- **リンク設置場所**: InstagramはリンクインバイオまたはストーリーのURLスタンプ（フォロワー数に関係なく利用可能）。TikTokはプロフィール欄のURLのみリンク設置可能
- **確定申告**: アフィリエイト収入は雑所得として申告が必要。報酬受取には各ASPへの口座登録が必須
