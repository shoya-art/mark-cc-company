# Instagram・TikTok プロフィール改善案

作成日: 2026-03-25
運営者: 復縁アドバイザー ジロー（Instagram: ziro_fukuen_pro / TikTok: @ziro__fukuen_master）

## 現状分析

| 指標 | 数値 |
|------|------|
| Instagramフォロワー | 1,713人 |
| Instagram投稿数 | 780本 |
| 週間リーチ | 3〜5万 |
| TikTokフォロワー | 4,089人 |
| TikTokいいね総数 | 54.2万 |
| プロフィール訪問率 | 約1.3%（ボトルネック） |
| 目標プロフィール訪問率 | 5%以上 |

### ボトルネックの構造的分析（確定版）

**現状確認済み情報：**
- リール内でプロフィール誘導CTAを入れている
- 1枚画像リール：平均再生時間7〜9秒、スキップ率42〜47%
- トークリール：平均再生時間19〜32秒、スキップ率38〜44%

**根本原因：CTAが視聴者に届いていない**

CTA自体は入れているが、ほとんどの視聴者が**CTAを見る前に離脱している**。

- 1枚画像リール（7〜9秒）：動画が短くCTAが末尾にある場合、スキップした47%には届かない
- トークリール（19〜32秒）：60秒以上の動画なら末尾のCTAは視聴者の半数以下にしか届かない

**解決策：CTAを冒頭に移動する**

## Instagram 改善案

---

### 1. プロフィール文（改善案）

#### 現状の問題（一般的なパターン）

- 名前欄がアカウント名のみで検索されにくい
- bioが「何者か」だけで「何が得られるか」が書かれていない
- LINE誘導の動線が弱い

#### 改善案

**名前欄（検索対策込み）**

```
復縁アドバイザー ジロー｜元カレと復縁する方法
```

> ポイント: Instagram検索は「名前欄」もインデックスされる。「復縁」「復縁する方法」等のキーワードを名前欄に入れることで検索流入が増える。

---

**bio文（140字以内）**

```
復縁成功者を100人以上サポートしてきた復縁アドバイザー
「また会いたい」と思わせる女性に変わる方法を発信中

✔ 元カレの心理・連絡タイミングの見極め
✔ 依存を手放してもう一度選ばれる方法
✔ 復縁後も続く関係の作り方

👇 今すぐ復縁診断を受けてみる（公式LINE）
```

> ポイント:
> - 実績（100人以上）を冒頭に置いて信頼性を担保する
> - 「自分が得られるもの」を3点箇条書きで明示する
> - CTAを最終行に集約し、LINE誘導を最優先にする

---

**リンク設置**

- リンクインバイオには「LINE登録ページ」または「Linktree」を設置
- Linktreeを使う場合の構成:
  1. 公式LINEに登録（最優先）
  2. 復縁診断（無料）
  3. おすすめ電話占い（アフィリエイトリンク）

---

### 2. リール CTA 改善案（5パターン）

プロフィール訪問率を上げるためには「動画の中」または「キャプション冒頭」でプロフィールへの誘導を明示的に行うことが最も効果的。

---

**パターン1（価値予告型）**

動画内セリフ or キャプション冒頭:

```
このアカウントでは「元カレに選ばれ直す方法」を毎日発信しています。
プロフィールから過去の投稿もチェックしてみてください。
```

> 狙い: 「もっと見たい」という気持ちをプロフィール訪問につなげる

---

**パターン2（LINE誘導型）**

```
無料の復縁診断をやっています。
プロフィールのリンクから受け取れます。
今の状況に合ったアドバイスを送ります。
```

> 狙い: 無料オファーを使ってプロフィールページに行く理由を作る

---

**パターン3（続き予告型）**

```
この続きはプロフィールのハイライトにまとめてあります。
「復縁ステップ」というところを見てみてください。
```

> 狙い: コンテンツの続きをプロフィールに置くことで訪問必然性を作る

---

**パターン4（ストーリー連動型）**

```
今日のテーマについてストーリーで詳しく話しました。
気になる方はプロフィールからフォローして見てください。
```

> 狙い: フォロー→ストーリー視聴という流れを作る

---

**パターン5（共感型）**

```
「私のことだ」と思った方、プロフィールを見てください。
同じ悩みを持つ方に向けた情報を毎日投稿しています。
```

> 狙い: コンテンツへの共感を、プロフィール訪問の動機に変換する

---

### 3. ハイライト構成案

ハイライトはプロフィールページで最初に目に入るコンテンツ。訪問者に「このアカウントを信頼できる」「フォローして得がある」と感じさせる構成にする。

| ハイライト名 | 内容 | 目的 |
|-------------|------|------|
| 復縁ステップ | 復縁成功までの5ステップをストーリー形式で解説 | 価値提供・信頼構築 |
| よくある失敗 | 復縁失敗パターントップ3・NG行動まとめ | 共感獲得・保存率UP |
| 成功事例 | サポートした方のビフォーアフター（匿名OK） | 社会的証明・信頼性 |
| 無料診断 | 復縁診断の案内・LINE誘導 | LINE登録促進 |
| ジローについて | 自己紹介・なぜ復縁アドバイザーになったか | 親近感・信頼構築 |
| 占い紹介 | おすすめ電話占いの紹介（アフィリエイト） | 収益化 |

> ポイント:
> - ハイライトカバー画像はアイコンを統一してビジュアル的に整えること（ブランディング）
> - 「無料診断」は左から2〜3番目に置き、目立つ位置をキープする

## TikTok 改善案

---

### 1. プロフィール文（改善案）

TikTokのプロフィールは80字程度が表示限界。短く・明確に・次の行動を指示する構成にする。

**現状の問題（一般的なパターン）**

- 何をしている人かが瞬時にわからない
- 視聴者が「フォローする理由」を判断できない
- URLリンクへの誘導が弱い

**改善案**

```
復縁アドバイザー ジロー
元カレと復縁したい女性を全力サポート
✔ 復縁成功者100人以上
✔ 元カレ心理・連絡タイミングを発信中
👇 無料復縁診断はプロフィールリンクへ
```

> ポイント:
> - 1行目で「何者か」を即座に伝える
> - 実績数字を入れて信頼性を確保する
> - 最終行で必ずCTAを入れ、リンクへ誘導する

**プロフィールリンク設置**

- TikTokプロフィールのURLは1つだけ設置可能
- LINE登録ページを直接設置するか、Linktree経由でLINE→占いリンクへ誘導する構成が最適

---

### 2. 動画内 CTA 改善案（5パターン）

TikTokは動画の最初の2〜3秒でスワイプされるかどうかが決まる。CTAは動画の「冒頭」または「最後5秒」に入れる。

---

**パターン1（冒頭フック + プロフィール誘導）**

```
【冒頭】
復縁できない人がやっている行動、知ってますか？
【最後】
他にも復縁のリアルな話をしています。
プロフィールから見てみてください。
```

---

**パターン2（LINE誘導型）**

```
【最後】
無料の復縁診断を公式LINEでやっています。
プロフィールのリンクから受け取れます。
今の状況を入力するだけでOKです。
```

---

**パターン3（続きシリーズ型）**

```
【最後】
このシリーズの続きもプロフィールにあります。
「復縁」で検索して過去動画も見てみてください。
```

---

**パターン4（コメント促進型）**

```
【最後】
「当てはまる」と思った方はコメントで教えてください。
プロフィールから過去の投稿も見られます。
```

> 狙い: コメントが増えると動画のリーチが広がり、プロフィール訪問者も増える

---

**パターン5（共感クローズ型）**

```
【最後】
今これを見ているあなたの状況が少しでも楽になれば嬉しいです。
もっと知りたい方はプロフィールをフォローしてください。
```

## 優先実施アクション（推奨順）

1. **Instagramプロフィール文の書き換え**（即日対応可・最も費用対効果が高い）
2. **ハイライトの整備**（特に「復縁ステップ」「無料診断」を先に作成）
3. **TikTokプロフィール文の書き換え**（即日対応可）

5. **Linktreeの整備**（LINE登録→占いリンクの導線を整える）
