# リール台本生成スキル

復縁コーチングInstagramアカウント（@mark_fukuen）向けのリール台本を生成する。

---

## 実際の台本から学んだ「絶対に守るルール」

### 文体・フォーマット
- **1行1〜5文字程度で改行する**（スマホ画面で読みやすくするため）
- 「。」より改行で区切る
- 体言止め・断言を多用する
- 漢字・ひらがなのバランスは実際の台本に合わせる（堅すぎない）
- 絵文字は一切使わない

### ボリューム
- 実際の台本は**400〜700文字程度**（短すぎると伝わらない）
- 冒頭フック：2〜4行
- 本文：リスト型は各項目3〜5行 × 3〜5個 / 感情型は段落を4〜6つ
- 締め：2〜4行
- CTA：3〜5行
- **合計で実際のリール30〜45秒読める量**を必ず確保する

### 安心感・納得感・目新しさの作り方
- **冒頭で「まず安心してください」「〜な人へ」など受け取り手に寄り添う**
- 「なぜそうなるのか」の理由を必ず説明する（納得感）
- 男性心理・行動パターンの「意外な真実」を1つ必ず入れる（目新しさ）
- 最後は「だから大丈夫」「諦めなくていい」で締める（安心感）
- 「実際にサポートして復縁できた女性の9割が〜」のような実績数字を入れると信頼度UP

### エンタメ構造（最重要・頭の片隅に常に置く）
- **共感は短く済ませる**。長い共感は離脱を招く。1〜3行で「そうそう」と思わせたら即切り上げる
- 共感の直後に**意外性・逆説・驚き**を入れて「え、どういうこと？」を作る
- 映画・小説の「それでどうなる？」構造を台本の流れに組み込む
  - 問いを立てる → すぐに答えない → 引っ張ってから明かす
  - 「実は〜」「でも、本当のことを言います」「ここからが本題です」のような転換ワードを使う
- 視聴者が次の行を読みたくなる「引き」を各段落の末尾に作る
- **テンポが命**。共感→意外性→納得→希望の流れを間延びさせない

### 主張先行型（新パターン・積極的に使う）
共感から入らず、いきなり「意外な事実・主張」で始める型。
離脱が早いリールに特に有効。

**構成：**
```
[タイトル]
断言 or 意外な事実

[冒頭：主張をいきなり出す]
「実は〜な男性って、〇〇なんです」
「〜する男性は、実は〜です」

[問いを立てる]
「では、なぜ〜なのか？」
「じゃあ、〜はどういうことか？」

[答えを展開する]
理由・背景を丁寧に説明
（ここで納得感・目新しさを作る）

[希望・結論]
「だから〜」「つまり〜」で安心感を作る

[CTA]
```

**例（既読無視テーマ）：**
```
既読無視する男性は
あなたのことを考えています

実は男性は、嫌いな相手のLINEは
既読すらしません。

既読をつけたということは
あなたのことは嫌いではないということ。

では、なぜ返事をしないのか？

それは〜（理由を展開）
```

## 3つのパターンと生成ルール

### パターンA：感情共感型
**いつ使う：** 拡散・いいね・フォロワー獲得が目的のとき

**構成：**
```
[タイトル]
「〜（主語）は〜（断言）」
意外性ワード必須（「絶対」「必ず」「一生」「運命」）

[冒頭フック]
視聴者の「今の状態」を言語化する
例：「まだ忘れられない人へ」「〜で悩んでいる人、読んで下さい」

[本文①：現状の言語化]
「でも、実際はどうですか？」の問いかけ
→ 視聴者が「そうそう！」となる具体的なシーン・感情を描く

[本文②：意外な男性心理・真実]
「実は〜なんです」「〜には理由があります」
→ 知らなかった視点を提供して「なるほど」を作る

[本文③：希望の提示]
今の状況をポジティブに再解釈する
「だからこそ〜」「それは〜のサインです」

[締め]
断言で安心させる
「あなたの気持ちは無駄じゃない」
「どうか〜と思わずに進んでいきましょう」
```

**CTA（目的別）：**
- 拡散重視 → 「共感した人は再投稿してください」
- フォロワー重視 → 「プロフィールから他の投稿もチェックしてください」
- LINE誘導 → 「彼と復縁する方法をテキストにまとめたのでこちらから受け取ってください」

---

### パターンB：知識リスト型
**いつ使う：** 保存数を増やしたいとき・教育コンテンツとして残したいとき

**構成：**
```
[タイトル]
「〜には〇つの〜があります」
「男性が〜する〇つのパターン」

[冒頭フック]
「5000人以上の復縁相談に乗ってきて〜」（実績の信頼性）
または「〜を知っていますか？今から解説します」

[本文：番号リスト × 3〜5個]
各項目：
① 見出し（インパクトある言葉）
具体的な場面描写（スマホに届いてた「お疲れさま」のLINEが来ない、など）
「その瞬間に〜を感じます」で締める

[締め]
リストを総括して希望につなぐ
「どんな別れ方をしても〜」

[CTA]
「保存していつでも見返せるようにして下さい」（必須）
```

---

### パターンC：シーン描写型
**いつ使う：** 視聴完了率を上げたいとき・認知拡大のとき

**構成：**
```
[タイトル]
「男性が〜する瞬間」
「元彼が〜とき、実は〜」

[冒頭フック]
具体的な場面から始める
「仕事から帰ってきてスマホを開いたら〜」
「共通の友達から元彼の話を聞いたとき〜」

[本文]
場面を追いながら感情・心理を解説する
「その時、彼は〜を感じています」
「これは〜だからです」（理由を必ず入れる）

[締め]
場面から普遍的な真実に広げる
「だから今、諦めなくていい理由があります」

[CTA]
「同じ経験がある人はいいねしてください」
または「保存して、落ち込んだときに読み返してください」
```

## テーマネタ帳

指定がない場合はここから選ぶ。

**状況系**
- 既読無視が続いている
- 自然消滅した
- 「友達に戻ろう」と言われた
- 突然ブロックされた
- 遠距離が原因で別れた
- 「好きな人ができた」と言われた
- 価値観の違いで別れた

**行動・サイン系**
- 元彼がSNSを消した
- 連絡頻度が急に減った
- 元彼がInstagramのフォローを外した
- 共通の友人経由で元彼の近況を聞いた

**感情系**
- まだ忘れられない
- 新しい人と会っても「なんか違う」
- 別れた後の方が彼の大切さがわかった
- 振られたのに諦められない

**復縁プロセス系**
- 冷却期間の正しい過ごし方
- 連絡再開のベストタイミング
- 告白するタイミングの見極め方
- やり直した後に失敗しないために

## 重要な注意事項

- 短すぎる台本は出力しない。実際の台本と同じ文字数・行数を確保する
- 「男性心理の意外な真実」を必ず1つ入れる（納得感・目新しさの核心）
- 冒頭で視聴者の現状を言語化して「これは自分のことだ」と思わせる（安心感の起点）
- CTAは毎回必ず入れる。「保存してください」または行動を促す一言で終わる
- **CTAは前後の脈絡なく唐突に出さない**。以下の3ステップで自然につなぐ
  1. 今の感情・状況をそのまま受け止める（「〜は本当につらいと思います」）
  2. それでも続く気持ちを引き出す（「それでもやっぱり〜と思う方は」）
  3. その感情の表明としてCTAを置く（「いいねで教えてください」）
- CTAは「エンゲージメントの依頼」ではなく「感情の表明」として機能させる
- 同じタイトルやフレーズの使い回しをしない（実績データで伸び悩みが確認済み）
