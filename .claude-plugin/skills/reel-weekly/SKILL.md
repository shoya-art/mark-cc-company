# リール台本 週次一括生成スキル

毎週日曜日に実行する。月曜〜日曜の7本分のリール台本を一括生成してファイルに保存する。

---

## 実行前に確認すること

1. **今日の日付を確認する**（ファイル名・投稿予定日に使う）
2. **既存の台本ファイルを確認する**（`sales/proposals/` または `marketing/content-plan/` 内のreelファイル）
3. **先週使ったテーマ・パターンを確認して重複・偏りを避ける**

---

## 7本のパターン配分ルール

1週間でパターンを分散させる。同じパターンが連続しないようにする。

| 曜日 | 推奨パターン | 理由 |
|------|------------|------|
| 月曜 | 主張先行型 | 週はじめ・スクロール中に目を止めやすい |
| 火曜 | 知識リスト型 | 保存数を稼ぐ・教育コンテンツ日 |
| 水曜 | 感情共感型 | 週中の疲れた感情に刺さる |
| 木曜 | 主張先行型 | 週後半の気分転換・意外性重視 |
| 金曜 | シーン描写型 | 週末前・視聴完了率重視 |
| 土曜 | 感情共感型 | 週末・感情が動きやすい |
| 日曜 | 知識リスト型 | 来週に向けて保存してもらう |

## テーマネタ帳

使用済みテーマと重複しないよう選定する。

**状況系**
- 既読無視が続いている
- 自然消滅した
- 「友達に戻ろう」と言われた
- 突然ブロックされた
- 遠距離が原因で別れた
- 「好きな人ができた」と言われた
- 価値観の違いで別れた
- 「もう好きかわからない」と言われた
- 喧嘩の後に連絡が来なくなった
- 彼から別れを切り出された

**行動・サイン系**
- 元彼がSNSを消した
- 連絡頻度が急に減った
- 元彼がフォローを外した
- 共通の友人から元彼の近況を聞いた
- 別れた後に元彼から突然連絡が来た
- 「元気？」だけ送ってきた
- 酔った勢いで連絡してきた
- 誕生日にだけ連絡してくる

**感情系**
- まだ忘れられない
- 新しい人と会っても「なんか違う」
- 別れた後の方が彼の大切さがわかった
- 振られたのに諦められない
- 追いLINEを送ってしまった後悔
- 感情的になってしまった後悔

**復縁プロセス系**
- 冷却期間の正しい過ごし方
- 連絡再開のタイミング
- 最初に送るLINEの内容
- 告白するタイミングの見極め
- やり直した後に失敗しないために
- 元彼が戻ってくるサイン

## パターン別生成ルール

### 主張先行型（月・木）

```
[タイトル]
断言 or 意外な事実（「〜する男性は〜です」形式）

[冒頭：主張をいきなり出す]
「実は〜な男性って、〇〇なんです」

[問いを立てる]
「では、なぜ〜なのか？」

[答えを展開する]
理由・背景を丁寧に説明（納得感・目新しさを作る）

[希望・結論]
「だから〜」で安心感を作る

[CTA]
感情を受け止める → まだ好きな気持ちを引き出す → いいね・保存を促す
```

### 感情共感型（水・土）

```
[タイトル]
「〜（主語）は〜（断言）」
意外性ワード必須（「絶対」「必ず」「一生」「運命」）

[冒頭]
視聴者の「今の状態」を1〜2行で言語化（短く済ませる）

[意外な転換]
「でも、実は〜」「これを知ってほしい」で即切り替える

[男性心理・意外な真実]
「なぜそうなるのか」の理由を説明（目新しさ・納得感）

[希望の提示]
「だから諦めなくていい」で安心感

[CTA]
感情を受け止める → まだ好きな気持ちを引き出す → いいね・保存
```

### 知識リスト型（火・日）

```
[タイトル]
「〜には〇つの〜があります」

[冒頭]
「5,000人以上の復縁相談に乗ってきて〜共通点がありました」

[本文：番号リスト × 3〜5個]
各項目：見出し + 具体的な場面描写 + 「その瞬間〜を感じます」で締める

[締め]
リストを総括して希望につなぐ

[CTA]
「後で見返せるように保存してください」
```

### シーン描写型（金）

```
[タイトル]
「男性が〜する瞬間」「元彼が〜とき、実は〜」

[冒頭]
具体的な場面から始める（スマホ・SNS・街中など）

[本文]
場面を追いながら心理を解説する
「その時、彼は〜を感じています」

[締め]
場面から普遍的な真実に広げる

[CTA]
「同じ経験がある人はいいねしてください」
```

## CTA生成ルール（最重要）

CTAは3ステップで作る。唐突に「いいねして」は絶対に出さない。

```
ステップ1：今の感情・状況をそのまま受け止める
「〜は本当につらいと思います」

ステップ2：それでも続く気持ちを引き出す
「それでもやっぱり彼のことが好きで」

ステップ3：感情の表明としてCTAを置く
「なんとか復縁したいと思う方はいいねで教えてください」
```

## 実行手順

1. 今日の日付を確認する
2. 既存の台本ファイルを確認して使用済みテーマを把握する
3. 7本分のパターン×テーマの組み合わせを選定する
4. 選定内容をユーザーに確認する（テーマ一覧を提示）
5. 承認が得られたら7本を順番に生成・1ファイルにまとめて保存する
6. 完了後に一覧を報告する

---

## 絶対に守るルール

- 1行1〜5文字程度で改行する（スマホ画面での読みやすさ）
- 文字数は400〜700文字（短すぎると伝わらない）
- 絵文字は一切使わない
- CTAは感情を受け止めてから自然につなぐ（唐突にいいね・保存を求めない）
- 同じテーマ・パターンが連続しないよう分散させる
- 全台本を完成状態で書き出す
